1. If Clauses (Si Cláusulas)
Type 1 (Real): Si + Present → Present / Future / Imperative
Type 2 (Hypothetical): Si + Past Subjunctive → Conditional
StructureExample
Type 1 (Present)Si llueve, no vamos a la playa.
(If it rains, we don’t go to the beach.)
Type 1 (Future)Si estudias, aprobarás el examen.
(If you study, you will pass the exam.)
Type 1 (Imp)Si tienes hambre, come algo.
(If you’re hungry, eat something.)
Type 2 (Cond)Si ellos fueran más inteligentes, no comprarían esta casa.
(If they were smarter, they wouldn't buy this house.)
Type 2 (Cond)Ellos vendrían a mi fiesta si sus padres les permitieran.
(They would come if their parents allowed.)

3. Reporting Questions
Yes/No: Use "si".
Open: Use question word.
DirectReported
¿Vendrás mañana?Me preguntó si vendría mañana.
(Asked if I would come.)
¿Dónde vives?Me preguntó dónde vivía.
(Asked where I lived.)
4. Reporting Commands
Use the Subjunctive mood.
DirectReported
"Haz tu tarea."Me dijo que hiciera mi tarea.
"No salgas tarde."Me dijo que no saliera tarde.
