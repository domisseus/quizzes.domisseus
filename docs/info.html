About geo.domisseus

About the Site

geo.domisseus is designed to help you practice GeoGuessr-related skills and learn geography more effectively. Right now, the site focuses on two main areas:

  • Cities:

    Practice locating major cities on an interactive map. You can only make three attempts per city, and it will only count as correct if you get it on your first try. After that, the correct location will be revealed.
    • World: Easy – 10 cities, Hard – 30 cities
    • Individual Countries: Easy – 3 cities, Hard – 10 cities
    • Custom: Not implemented yet – User can type amount of cities to practice. Default: All important major cities (the ones that appear bold on Google Maps)
    • Small Nations: All countries with 1-3 important cities

  • Learning Language Alphabets: (SOON)

    This feature helps you get familiar with different scripts and alphabets around the world
